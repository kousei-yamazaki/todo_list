# ToDo管理アプリ

シンプルで使いやすい、Windows向けのローカルToDo管理アプリケーションです。  
インターネット接続不要で、データはすべてお使いのPC内に保存されます。

---

## 動作環境

| 項目 | 内容 |
|------|------|
| OS | Windows |
| Python | 3.x（標準ライブラリのみ使用） |
| ネットワーク | 不要（オフライン動作） |

---

## 起動方法

### バッチファイルから起動（推奨）

`run_app.bat` をダブルクリックしてください。

### コマンドラインから起動

```
python main.py
```

---

## 使い方

### タスクを追加する

1. 画面上部の **「＋ タスクを追加」** ボタンをクリックします。
2. **タイトル**（必須）と **メモ**（任意）を入力します。
3. **「保存」** ボタンをクリックするか、`Enter` キーを押すと追加されます。

> キャンセルする場合は「キャンセル」ボタンまたは `Esc` キーを押してください。

---

### タスクを完了にする

タスク左側の **チェックボックス** をクリックすると、完了／未完了を切り替えられます。

- ✅ チェックを入れると「完了タスク」欄に移動します。
- チェックを外すと「未完了タスク」欄に戻ります。

---

### タスクを編集する

1. 編集したいタスクの **「編集」** ボタンをクリックします。
2. タイトルやメモを修正します。
3. **「保存」** ボタンをクリックして変更を確定します。

---

### タスクを削除する

1. 削除したいタスクの **「削除」** ボタンをクリックします。
2. 確認ダイアログが表示されます。
3. **「はい」** を選択すると削除されます。

---

## 画面の見方

```
┌─────────────────────────────────────────┐
│  ＋ タスクを追加                         │  ← ツールバー
├─────────────────────────────────────────┤
│  📋 未完了タスク                         │
│  ☐  買い物をする        [編集] [削除]    │
│  ☐  レポートを提出する  [編集] [削除]    │
│─────────────────────────────────────────│
│  ✅ 完了タスク                           │
│  ☑  本を読む            [編集] [削除]    │
└─────────────────────────────────────────┘
```

| 要素 | 説明 |
|------|------|
| 📋 未完了タスク | まだ完了していないタスクの一覧 |
| ✅ 完了タスク | 完了済みのタスクの一覧 |
| チェックボックス | クリックで完了／未完了を切り替え |
| 編集ボタン | タスクのタイトル・メモを編集 |
| 削除ボタン | タスクを削除（確認あり） |

---

## データの保存について

- タスクのデータは操作のたびに自動で保存されます。手動での保存操作は不要です。
- データは `data/tasks.json` にローカル保存されます。
- アプリを閉じてもデータは保持されます。
